# Dice Roller

Roll dice

## Description

a simple project to simulate dice rolls from dice notation
## Getting Started

### Dependencies

* Java 8
* Javac

### Installing

* Download the GitHub repo

### Executing program

* open cmd
* navigate to the directory of the install
* run with java
```
{yourInstallPath}\diceRoller\src> java diceRoller.java
```
